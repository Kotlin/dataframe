# Exposed Example

Showcase of using Kotlin DataFrame with [Exposed](https://github.com/JetBrains/Exposed).

Even though Exposed is not officially supported as a data source in DataFrame,
this project shows how to convert from and to Exposed tables and what else is possible combining
the two.

This project uses the
[Kotlin DataFrame Compiler Plugin](https://kotlin.github.io/dataframe/compiler-plugin.html).

We recommend using an up-to-date IntelliJ IDEA for the best experience,
as well as the latest Kotlin plugin version.

> [!WARNING]
> For proper functionality in IntelliJ IDEA requires version 2025.2 or newer.

[Download this Example](https://github.com/Kotlin/dataframe/raw/example-projects-archives/exposed.zip)
