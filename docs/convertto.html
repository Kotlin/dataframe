Dataframe 0.8 Help

convertTo

Converts columns in DataFrame to match given schema.

convertTo<T>()

Any additional columns will be dropped.

You can provide custom converters and parsers:

class IntClass(val value: Int) @DataSchema class IntSchema(val ints: IntClass) val df = dataFrameOf("ints")(1, 2, 3) df.convertTo<IntSchema> { convert<Int>().with { IntClass(it) } }
